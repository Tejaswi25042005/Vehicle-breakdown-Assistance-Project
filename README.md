# 🚗 Vehicle Breakdown Assistance System

A web-based application developed to provide emergency vehicle assistance services.  
The system allows users to request roadside help and enables admin and driver management for handling service requests efficiently.

---

## 🔧 Technologies Used

- PHP
- MySQL
- HTML
- CSS
- JavaScript
- Bootstrap (for UI styling)

---

## ✨ Key Features

- User Registration and Login
- Book Vehicle Assistance Request
- Admin Dashboard
- Driver Management System
- Request Status Tracking
- Secure Database Integration

---

## 🗄 Database

The database file included in this repository:

vehassistancemsdb.sql

Import this file into MySQL to run the project locally
